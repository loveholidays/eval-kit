# eval-kit

TypeScript SDK for content evaluation

## Installation

```bash
pnpm install eval-kit
```

## Development

### Setup

```bash
# Install dependencies
pnpm install
```

### Scripts

```bash
# Build the project
pnpm build

# Run tests
pnpm test

# Run tests in watch mode
pnpm test:watch

# Run tests with coverage
pnpm test:coverage

# Lint and check code
pnpm lint

# Lint and fix issues
pnpm lint:fix

# Format code
pnpm format

# Type check
pnpm typecheck
```

## Project Structure

```
eval-kit/
├── src/          # Source code
├── tests/        # Test files
├── dist/         # Built output (generated)
└── coverage/     # Test coverage reports (generated)
```
